BrightMark Agency CRM Dashboard
UI/UX Design Task 3 — Future Interns 2026

About
BrightMark Agency is a 5-member digital marketing agency managing 30+ clients across SEO, Social Media, and Paid Ads. This CRM dashboard was designed to solve their two biggest
problems: missed follow-ups and no pipeline visibility.


Workflow Logic

The dashboard follows the natural agency workflow:

Lead Comes In → Added to Pipeline → Contacted → 
Proposal Sent → Negotiation → Converted to Client → 
Tasks Assigned → Follow-ups Tracked → Client Managed

Each screen maps to one step in this workflow:
- Dashboard = Daily overview of entire agency
- Lead Pipeline = Track every lead across 6 stages
- Client List = Manage all active clients
- Client Profile = Deep view of individual client
- Task Tracker = Never miss a follow-up


UX Decisions

1. Dark Sidebar
   Separates navigation from content clearly.
   Active page is highlighted in purple.

2. Color Coded Status System
   Green = Active / Good
   Red = Overdue / Lost / Inactive
   Amber = At Risk / Due Today
   Blue = New / Upcoming
   Same colors used across all 5 screens.

3. Kanban Pipeline
   Shows all lead stages at a glance.
   Owner can instantly spot bottlenecks.

4. Summary Cards on Dashboard
   4 key numbers visible immediately.
   No clicking needed to understand agency health.

5. Overdue Tab in Task Tracker
   Placed second in tab bar for quick access.
   Ensures team addresses overdue tasks first.

How This Helps Agencies Work Efficiently

- Owner sees full agency status in under 60 seconds
- No lead gets forgotten — pipeline shows every stage
- No follow-up missed — task tracker shows overdue items
- At-risk clients identified early through status tags
- Team members know their responsibilities clearly
- Layout scales from 30 to 100+ clients easily

Screens Designed
1. Dashboard — Daily overview with summary cards
2. Lead Pipeline — Kanban board with 6 stages
3. Client List — Searchable table of all clients
4. Client Profile — Individual client detail view
5. Task Tracker — Follow-up and task management
